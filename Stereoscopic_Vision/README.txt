Les touches décrites ci-dessous sont également décrite dans le menu "Help"

-- Commandes Clavier : 

* Key Up/Down/Left/Right : Move around
* Mouse mouve : Move camera
* Key P/M : Change fov
* Key U/D : Go up and down
* Key Space : Jump
* Key R : Run
* Key S : Switch between view
* Key H : Help Menu
* Key E : Quit 


-- Commandes Manette de XBox :

* Joystick Left : Move around
* Joystick Right : Move camera
* Button Up/Down : Change fov
* Button X/Y : Go up and down
* Button A : Jump
* Button L3 + Joystick : Run
* Button Start :Switch between view
* Button B : Help Menu
* Button back : Quit

----------------

Bonus : 

- Possibilité de jouer via le clavier ainsi que la manette de XBox.  
- En vue "Double Rendu" (Partie 4), il est possible de cliquer sur les boutons présents sur l'image. 
(Flèche : exit/ Engrenage : help)
- Possibilité d'alterner entre les différentes vues en appuyant sur le bouton S (clavier) ou Start (manette)
- Possibilité de se baisser/accroupir et de se relever dans un certain range. On ne peut donc pas aller plus haut que la position initiale de la caméra, mais il est possible de se baisser jusqu'au niveau du sol. (Touches U/D sur le clavier et X/Y sur la manette).
- Possibilité de courir en appuyant sur R sur le clavier ou bien appuyant sur le joystick gauche de la manette de XBox.
- Possibilité de sauter en appuyant sur la touche espace du clavier ou bien le bouton A sur la manette.
